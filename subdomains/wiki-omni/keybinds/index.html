Essential shortcuts

The buttons players will reach for constantly during normal play.

Core loop
Move

Standard movement keys for exploration, positioning, and combat.

WASD
Sprint

Hold to move faster across open ground.

Shift
Jump / Vault

Use for jumps, ledges, and low obstacles.

Space
Interact

Talk to characters, pick things up, and use doors or camp stations.

E
Open inventory

Check gear, quick slots, and important items.

Tab
Quick map

Open the current map and your active objective view.

M
Utility action

Use the tool or context action you have equipped.

Q
Pause / Back

Open the pause menu or back out of the screen you are in.

Esc

Combat & field actions

Less frequent, but still worth finding fast when things get messy.

Situational
Primary action

Use your main attack, tool, or equipped action.

LMB
Secondary action

Use an alternate fire mode, aim state, or secondary tool.

RMB
Swap tool

Cycle quick-slot tools without opening the inventory.

MW

Toggle the support action used for visibility or clue tracking.

Using this page

This page is here for speed. You should be able to glance at it once and get straight back to playing.

If a key matters during normal play, it belongs near the top. If it only matters once in a while, it can sit lower down.

  • Keep movement and interaction together so the core loop is easy to scan.
  • Only add extra description when the action is easy to forget under pressure.
  • If controller support gets added later, mirror this layout instead of mixing formats.
